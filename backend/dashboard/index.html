Testing shortcuts

Use these flows to quickly validate the backend from the browser or your test runner.

Run all pytest

Runs the full Python test suite under backend/tests. Use this when you want the broadest backend check.

Run extended APIs

Runs tests/test_api_extended.py, which covers extra routes like dashboard, VIP, cosmetics, push, and guilds.

Run engagement suite

Runs tests/test_engagement_phases.py to verify missions, battle pass, collection, streaks, and live-game flows.

Run smoke script

Runs scripts/smoke_test_apis.py against the live API for a fast end-to-end sanity check.

Fast smoke checks

  1. Open /api/dashboard.
  2. Check /api/dashboard/data for the shape.
  3. Use /api/guest/login and /api/user/{user_id}.
  4. Try /api/rooms and /api/computer-match.

Useful validation

  1. Daily reward and streak info.
  2. Battle pass and collection progress.
  3. Razorpay config + mock order/verify.
  4. Guild create / list / join / leave.

Live multiplayer

  1. Create a room.
  2. Join from a second user.
  3. Open the WebSocket room URL.
  4. Start and claim bingo.
